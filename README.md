# app-fdr

## Installation

=======
This guide assumes a fresh install.

Node
[https://nodejs.org/en/](Node)

Yeoman
```
$ npm install -g yo
```

Bower
```
$ npm install -g bower
```

Gulp
```
$ npm install -g gulp
```

Yeoman es6 Angular Generator
(Should install es6 deps, ie: Babel)
```
$ npm install -g generator-es6-angular
```

Sass
```
$ gem install sass
```
(If error)

```
$ sudo gem install sass
```

Install deps
```
$ npm i
$ bower i
```

## Run
$ gulp serve
